AInomics Japan

金融・経済ニュースを日本語でわかりやすく解説するショート動画を、AIを活用して制作・配信するコンテンツパブリッシングサービスです。

サービスについて

AInomics Japanは、米国・日本の市況ニュースやテクノロジー企業の動向を、最新の情報を基に 日本語の短尺動画として継続的に企画・生成し、YouTube ShortsおよびTikTokに配信するコンテンツ パブリッシングの仕組みです。配信内容は速報解説・企業紹介・AIと経済をテーマにした解説など、 複数のフォーマットで構成されています。

市況ニュース解説

米国・日本の株式市場の動きを、背景にある経済的な矛盾や逆説を交えて解説

企業スポットライト

上場前の注目スタートアップを、資金調達・技術内容とともに紹介

モーニングブリーフ

前日の市場の終値・主要指標を朝にまとめて配信

AI×経済

AIが経済・産業に与える影響をテーマにした解説コンテンツ

すべてのコンテンツは情報提供・教育目的であり、投資助言には該当しません。投資判断はご自身の責任において行ってください。

仕組み

配信される動画は、以下のプロセスを経て継続的に制作・公開されています。

1
公開されているニュース・市場データを収集し、配信テーマを選定します。
2
収集した情報のみを基に、日本語の解説台本を生成します。
3
台本の事実関係・表現を複数の工程でチェックした上で、音声・字幕付きの動画を生成します。
4
YouTube Data API および TikTok Content Posting API を通じて、各プラットフォームに配信します。
