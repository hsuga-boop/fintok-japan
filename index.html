AInomics Japan

金融・経済コンテンツクリエイター向けの、AIによる動画生成・自動配信プラットフォームです。TikTokおよびYouTubeへの継続的な動画配信を支援します。

サービスについて

AInomics Japanは、金融・経済分野のコンテンツクリエイターを対象に、AIを活用した動画コンテンツの 自動生成・配信プラットフォームを提供するサービスです。登録クリエイターのTikTokおよびYouTube チャンネルに対して、最新の市場ニュースや経済解説を短尺動画として継続配信します。

市況ニュース解説

米国・日本の株式市場の動きを、経済的な背景とともに解説

企業スポットライト

注目スタートアップを資金調達・技術内容とともに紹介

モーニングブリーフ

前日の市場終値・主要指標をまとめた朝の解説動画

AI×経済

AIが経済・産業に与える影響をテーマにした解説コンテンツ

すべてのコンテンツは情報提供・教育目的であり、投資助言には該当しません。投資判断はご自身の責任において行ってください。

サービスの仕組み

クリエイターがサービスに登録すると、以下のプロセスで動画が自動生成・配信されます。

1
クリエイターがサービスに登録し、配信チャンネル（TikTok / YouTube）を連携します。
2
公開されているニュース・市場データを収集し、クリエイターのチャンネルに最適なテーマを選定します。
3
日本語の解説台本を生成し、事実確認・表現チェックを経て音声・字幕付きの動画を生成します。
4
YouTube Data API および TikTok Content Posting API を通じて、クリエイターのチャンネルに動画を配信します。
