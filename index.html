FinTok Japan プライバシーポリシー

最終更新日: 2026年6月14日

1. はじめに

FinTok Japan（以下「本アプリ」）は、YouTube Data API v3 を使用してショート動画を自動投稿する個人運営のコンテンツ自動化ツールです。 本アプリは Google API Services を利用しており、 Google API サービスユーザーデータポリシー （Limited Use ポリシーを含む）に準拠しています。

2. 収集するデータ

本アプリが Google アカウントから取得するデータは以下のとおりです：

  • YouTubeチャンネルの動画アップロード権限（youtube.upload）
  • YouTubeチャンネルの統計情報・動画再生数・コメント（youtube.readonly）

これらのデータはチャンネル運営者自身（本アプリの所有者）のみが使用します。第三者への提供・販売は行いません。

3. データの利用目的

  • YouTube Shortsへの動画の自動アップロード
  • 再生数・いいね数・コメントの収集（コンテンツ改善目的）
  • チャンネル登録者数の推移記録

取得したデータは、コンテンツ戦略の改善のみに使用し、分析・広告・第三者サービスへの転送には使用しません。

4. データの保存と保護

  • 認証トークンはローカル環境（youtube_tokens.json）に保存し、パーミッション 600（所有者のみ読み書き可）で保護します。
  • 統計データはクラウドデータベース（Supabase）に保存し、本アプリの所有者のみアクセス可能です。
  • データは7日〜90日間保持した後、自動削除されます。

5. Google ユーザーデータの限定使用

本アプリによる Google ユーザーデータの使用は、ユーザーへのサービス提供に必要な範囲に限定されます。 取得したデータを人間が読むこと、広告への利用、クレジットスコアへの利用、その他 Google API サービスポリシーで禁止された目的には使用しません。

6. データの削除

データの削除を希望する場合は、下記の連絡先までお問い合わせください。 YouTubeとの連携を解除するには、 Googleアカウントのアクセス権限ページ から本アプリを削除できます。
