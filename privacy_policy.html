AInomics Japan プライバシーポリシー

最終更新日: 2026年7月2日

1. はじめに

AInomics Japan（以下「本サービス」）は、金融・経済コンテンツクリエイター向けに、 AIを活用した動画生成・自動配信を提供するプラットフォームです。 本サービスは YouTube Data API v3 および TikTok Content Posting API を使用して、 登録クリエイターのチャンネルへ動画を配信します。 本サービスは Google API Services を利用しており、 Google API サービスユーザーデータポリシー （Limited Use ポリシーを含む）に準拠しています。 また、TikTok API の利用にあたっては TikTokプライバシーポリシー および TikTok のデベロッパー利用規約に準拠しています。

2. 収集するデータ（Google / YouTube）

本サービスが登録クリエイターの Google アカウントから取得するデータは以下のとおりです：

  • YouTubeチャンネルの動画アップロード権限（youtube.upload）
  • YouTubeチャンネルの統計情報・動画再生数・コメント（youtube.readonly）

これらのデータは、当該クリエイターへのサービス提供（動画配信・統計分析）の目的にのみ使用します。第三者への提供・販売は行いません。

3. 収集するデータ（TikTok）

本サービスが TikTok アカウントから取得するデータは、TikTok Content Posting API を通じて以下のスコープに限定されます：

  • user.info.basic：投稿先アカウントの識別に必要な基本プロフィール情報（表示名・アイコン等）
  • video.upload / video.publish：本サービスが生成した動画を登録クリエイターのTikTokアカウントへアップロード・公開するための権限

TikTok経由で取得した情報は、クリエイターの動画配信という本来の目的にのみ使用し、広告・第三者への提供・販売・分析目的での共有には使用しません。

4. データの利用目的

  • 登録クリエイターのYouTube ShortsおよびTikTokチャンネルへの動画の自動アップロード
  • 再生数・いいね数・コメントの収集（コンテンツ改善目的）
  • チャンネル登録者数の推移記録

取得したデータは、当該クリエイターへのサービス提供とコンテンツ戦略の改善のみに使用します。分析・広告・第三者サービスへの転送には使用しません。

5. データの保存と保護

  • 認証トークン（YouTube・TikTok共通）はセキュアな環境に保存し、クリエイターごとにアクセスを制限します。
  • TikTokのアクセストークン・リフレッシュトークンは暗号化して保存し、外部サーバーへの送信や第三者との共有は行いません。
  • 統計データはクラウドデータベース（Supabase）に保存し、当該クリエイターおよびサービス管理者のみアクセス可能です。
  • データは7日〜90日間保持した後、自動削除されます。

6. Google ユーザーデータの限定使用

本サービスによる Google ユーザーデータの使用は、クリエイターへのサービス提供に必要な範囲に限定されます。 取得したデータを人間が閲覧すること、広告への利用、クレジットスコアへの利用、その他 Google API サービスポリシーで禁止された目的には使用しません。

7. データの削除

データの削除を希望するクリエイターは、下記の連絡先までお問い合わせください。 YouTubeとの連携を解除するには、 Googleアカウントのアクセス権限ページ から本サービスを削除できます。 TikTokとの連携を解除するには、TikTokアプリ内の「設定とプライバシー」→「セキュリティ」→「連携しているアプリ」から本サービスのアクセスを取り消すことができます。 連携解除後、本サービスが保持しているTikTok関連の認証情報は速やかに削除されます。
