AInomics Japan プライバシーポリシー

最終更新日: 2026年6月26日

1. はじめに

AInomics Japan（以下「本アプリ」）は、YouTube Data API v3 および TikTok API を使用してショート動画を配信するコンテンツパブリッシングサービスです。 本アプリは Google API Services を利用しており、 Google API サービスユーザーデータポリシー （Limited Use ポリシーを含む）に準拠しています。 また、TikTok API の利用にあたっては TikTokプライバシーポリシー および TikTok のデベロッパー利用規約に準拠しています。

2. 収集するデータ（Google / YouTube）

本アプリが Google アカウントから取得するデータは以下のとおりです：

  • YouTubeチャンネルの動画アップロード権限（youtube.upload）
  • YouTubeチャンネルの統計情報・動画再生数・コメント（youtube.readonly）

これらのデータはチャンネル運営者自身（本アプリの所有者）のみが使用します。第三者への提供・販売は行いません。

3. 収集するデータ（TikTok）

本アプリが TikTok アカウントから取得するデータは、TikTok Login Kit / Content Posting API を通じて以下のスコープに限定されます：

  • user.info.basic：投稿先アカウントの識別に必要な基本プロフィール情報（表示名・アイコン等）
  • video.upload / video.publish：本アプリが生成した動画をTikTokアカウントへアップロード・公開するための権限

TikTok経由で取得した情報は、動画の自動投稿という本来の目的にのみ使用し、広告・第三者への提供・販売・人間による閲覧には使用しません。 TikTokのコンテンツ（動画・テキスト等）を分析目的で第三者と共有することもありません。

4. データの利用目的

  • YouTube ShortsおよびTikTokへの動画の自動アップロード
  • 再生数・いいね数・コメントの収集（コンテンツ改善目的）
  • チャンネル登録者数の推移記録

取得したデータは、コンテンツ戦略の改善のみに使用し、分析・広告・第三者サービスへの転送には使用しません。

5. データの保存と保護

  • 認証トークン（YouTube・TikTok共通）はローカル環境に保存し、パーミッション 600（所有者のみ読み書き可）で保護します。
  • TikTokのアクセストークン・リフレッシュトークンも同様にローカル環境のみに保存し、外部サーバーへの送信や第三者との共有は行いません。
  • 統計データはクラウドデータベース（Supabase）に保存し、本アプリの所有者のみアクセス可能です。
  • データは7日〜90日間保持した後、自動削除されます。

6. Google ユーザーデータの限定使用

本アプリによる Google ユーザーデータの使用は、ユーザーへのサービス提供に必要な範囲に限定されます。 取得したデータを人間が読むこと、広告への利用、クレジットスコアへの利用、その他 Google API サービスポリシーで禁止された目的には使用しません。

7. データの削除

データの削除を希望する場合は、下記の連絡先までお問い合わせください。 YouTubeとの連携を解除するには、 Googleアカウントのアクセス権限ページ から本アプリを削除できます。 TikTokとの連携を解除するには、TikTokアプリ内の「設定とプライバシー」→「セキュリティ」→「連携しているアプリ」から本アプリのアクセスを取り消すことができます。 連携解除後、本アプリが保持しているTikTok関連の認証情報は速やかに削除されます。
