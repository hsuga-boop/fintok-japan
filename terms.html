AInomics Japan 利用規約

最終更新日: 2026年7月2日

1. はじめに

AInomics Japan（以下「本サービス」）は、金融・経済コンテンツクリエイター向けに、 AIを活用した動画生成・自動配信プラットフォームを提供するサービスです。 登録クリエイターのTikTokおよびYouTubeチャンネルへの動画配信を自動化します。 本サービスを利用することで、以下の規約に同意したものとみなします。

2. サービスの目的

本サービスは以下を目的としています：

  • 金融・経済コンテンツクリエイターへの動画生成・自動配信機能の提供
  • 登録クリエイターのYouTube Shorts および TikTok チャンネルへの継続的な動画配信
  • 視聴者の金融リテラシー向上に資する教育的コンテンツの普及

3. 免責事項

本サービスが提供するコンテンツは、情報提供・教育目的のみであり、 投資助言・金融アドバイスには該当しません。 コンテンツを参考にした投資判断による損益について、本サービスは一切の責任を負いません。 投資判断は必ずご自身の責任において行ってください。

4. TikTok API の利用

本サービスは TikTok API を使用して動画を投稿します。 TikTok の 利用規約 および プライバシーポリシー に準拠して運営されます。 TikTok API 経由で収集されるデータは動画投稿の目的にのみ使用し、第三者への提供は行いません。

5. YouTube API の利用

本サービスは YouTube Data API v3 を使用して動画を投稿します。 YouTubeの利用規約 および Google API サービスユーザーデータポリシー に準拠して運営されます。

6. 知的財産権

本サービスが制作・投稿するすべてのコンテンツ（動画・テキスト・音声）の著作権は AInomics Japan に帰属します。 無断転載・二次利用はお断りします。

7. 規約の変更

本規約は予告なく変更されることがあります。 重要な変更がある場合はこのページに反映します。 継続してサービスを利用することで、変更後の規約に同意したものとみなします。
